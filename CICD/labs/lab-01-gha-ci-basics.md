# Lab 01: GitHub Actions CI Basics
Difficulty: Easy
Time: 45 minutes

## Objective
- Build and test a simple app on pull requests.

## Prerequisites
- GitHub repo with a Node.js or Python app

## Steps
1) Create `.github/workflows/ci.yml`.
2) Add build and test steps.
3) Open a PR to trigger the workflow.

## Validation
- Workflow runs and reports success on the PR.

## Cleanup
- Remove the workflow file if needed.
